# 必要パーツ

- CPU（中央演算処理装置）：パソコンの「頭脳」です。
- マザーボード：すべてのパーツを繋ぐ「基盤」です。
- メモリ (RAM)：データを一時的に置く「作業机」です。
- ストレージ (SSD/HDD)：データを保存する「本棚」です。現在は高速なM.2 SSDが主流です。
- 電源ユニット (PSU)：各パーツに電力を供給します。
- ケース：パーツを収める箱です。
- OS (Windowsなど)：パソコンを動かすための基本ソフトです。
- CPUクーラー：CPUを冷やします（CPUに付属している場合もあります）
